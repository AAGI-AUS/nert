# Package index

## COG Readers

Functions to read Cloud Optimised GeoTIFF data from TERN.

- [`read_tern()`](https://aagi-aus.github.io/nert/reference/read_tern.md)
  : Read TERN COG Datasets
- [`read_asc()`](https://aagi-aus.github.io/nert/reference/read_asc.md)
  : Read ASC COGs from TERN

## Utilities

Helper functions for working with TERN data.

- [`get_key()`](https://aagi-aus.github.io/nert/reference/get_key.md) :
  Get or Set Up API Key for TERN
- [`reexports`](https://aagi-aus.github.io/nert/reference/reexports.md)
  [`autoplot`](https://aagi-aus.github.io/nert/reference/reexports.md) :
  Objects exported from other packages

## Data

Example datasets bundled with the package.

- [`grain`](https://aagi-aus.github.io/nert/reference/grain.md) :
  Synthetic dataset for a grain production experiment in South Australia
