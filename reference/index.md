# Package index

## COG Readers

Functions to read Cloud Optimised GeoTIFF data from TERN.

- [`read_tern()`](https://aagi-aus.github.io/nert/reference/read_tern.md)
  : Read TERN COG Datasets
- [`read_smips()`](https://aagi-aus.github.io/nert/reference/read_smips.md)
  : Read TERN SMIPS Soil Moisture Data
- [`read_asc()`](https://aagi-aus.github.io/nert/reference/read_asc.md)
  : Read Australian Soil Classification (ASC) Data
- [`read_aet()`](https://aagi-aus.github.io/nert/reference/read_aet.md)
  : Read CMRSET Actual Evapotranspiration Data
- [`read_slga()`](https://aagi-aus.github.io/nert/reference/read_slga.md)
  : Read SLGA Soil and Landscape Grid of Australia Attributes
- [`read_soil_diversity()`](https://aagi-aus.github.io/nert/reference/read_soil_diversity.md)
  : Read Soil Beta Diversity (Bacteria and Fungi)
- [`read_canopy_height()`](https://aagi-aus.github.io/nert/reference/read_canopy_height.md)
  : Read Canopy Height Data (OzTreeMap)
- [`read_phenology()`](https://aagi-aus.github.io/nert/reference/read_phenology.md)
  : Read Land Surface Phenology Data (MODIS)

## Vectorised Collection

Batch extraction of TERN raster data at point locations.

- [`collect_tern_data()`](https://aagi-aus.github.io/nert/reference/collect_tern_data.md)
  : Collect TERN Data Over Time and Space

## Utilities

Helper functions for working with TERN data.

- [`get_key()`](https://aagi-aus.github.io/nert/reference/get_key.md) :
  Get or Set Up API Key for TERN
- [`reexports`](https://aagi-aus.github.io/nert/reference/reexports.md)
  [`autoplot`](https://aagi-aus.github.io/nert/reference/reexports.md) :
  Objects exported from other packages

## Data

Example datasets bundled with the package.

- [`grain`](https://aagi-aus.github.io/nert/reference/grain.md) :
  Synthetic dataset for a grain production experiment in South Australia
