# Internal handler for retrieving Canopy Height data

Internal handler for retrieving Canopy Height data

## Usage

``` r
.read_tern_canopy_height(api_key, max_tries, initial_delay)
```

## Arguments

- api_key:

  URL-encoded API key.

- max_tries, initial_delay:

  Passed to
  [`.read_cog()`](https://aagi-aus.github.io/nert/reference/dot-read_cog.md).
